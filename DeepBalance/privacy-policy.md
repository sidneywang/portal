# 隐私政策

**最后更新日期**：2026年6月10日

DeepBalance（以下简称"本应用"）尊重并保护所有使用者的个人隐私。本隐私政策旨在向你说明我们如何收集、使用、存储及保护你的信息。

## 1. 信息收集

本应用不收集任何个人身份信息。

- 本应用不需要创建账号，不需要登录，不收集你的姓名、邮箱、手机号等任何可识别身份的信息。
- 本应用不收集使用数据，不记录你的打开次数、使用时长、点击行为等统计信息。
- 本应用不收集位置信息，不请求任何位置权限。

## 2. 信息使用

由于本应用不收集任何信息，因此不存在使用、分析、分享或出售你的个人信息的情况。

## 3. 本地权限说明

为实现核心功能（通过菜单栏查看 DeepSeek 余额），本应用可能需要请求以下系统权限：

| 权限 | 用途 | 是否必需 |
|------|------|----------|
| 通知 / Widget | 用于在 Widget 中显示余额信息 | 否 |

你可以在系统设置中随时关闭这些权限，关闭后 Widget 功能将无法使用，但不影响其他功能。

## 4. 第三方服务

本应用不集成任何第三方 SDK（如友盟、Firebase、Google Analytics 等），不向任何第三方发送数据。

## 5. 数据存储

- 你输入的 API Key 仅保存在你的设备本地应用共享容器中，不会上传到任何服务器。
- 应用通过 DeepSeek 官方 API（api.deepseek.com）查询余额，查询过程中 API Key 仅在请求头中传递，不会被记录或存储。

## 6. 儿童隐私

本应用不会故意收集任何年龄用户的信息，包括 13 岁以下儿童。

## 7. 隐私政策变更

如果本隐私政策发生变更，我们会在应用更新时通过更新本文档的方式通知你。变更后的政策将在发布之日起生效。
